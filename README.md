# WebAPI_2.DAL

This is the Data Access Layer (DAL) for the Book Library API system.

- **Used by**: [WebAPI_2](https://github.com/Oleksandr-Zabo/WebAPI_2)
- **Purpose**: Provides Entity Framework Core models and database access logic.

## Usage
- This project is referenced by the main API project ([WebAPI_2](https://github.com/Oleksandr-Zabo/WebAPI_2)).
- No need to run separately; it is built and published together with the API.
